# Farivar Hypermarket store

A grocery store web application built with Flask and SQLite.

## Features

- Responsive mobile-first design
- Product detail pages
- SQLite database integration
- Admin panel
- Add products with image upload
- Delete products
- Dynamic product rendering
- Responsive navbar with hamburger menu

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript

## Setup Instructions

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/local-food-store.git
```

### Create virtual environment

```bash
python -m venv venv
```

### Activate virtual environment

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install flask
```

### Run the application

python app.py
```
